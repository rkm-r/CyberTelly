de~vlcInitError~Fehler beim Öffnen des VLC-Players
de~vlcInitError~----------------------------------
de~vlcInitError~Mögliche Fehlerquellen:
de~vlcInitError~  Alle Betriebssysteme:
de~vlcInitError~    VLC-Player ist nicht installiert
de~vlcInitError~  Linux:
de~vlcInitError~    VLC wurde vom Snap-Store oder Flathub
de~vlcInitError~    installiert und läuft in einer Sandbox:
de~vlcInitError~    Der Zugriff auf die VLC-Bibliothek ist 
de~vlcInitError~    deshalb blockiert.
de~vlcInitError~    Lösung:
de~vlcInitError~      1. VLC deinstallieren
de~vlcInitError~      2. Terminalfenster öffnen
de~vlcInitError~      3. Befehl: sudo apt install vlc
de~vlcInitError~         (VLC wird aus Repository installiert)
de~vlcInitError~      4. CyberTelly neu starten
de~vlcInitError~      5. Testen

en~waylandError~CyberTelly: Fatal error occurred
en~waylandError~
en~waylandError~System is using session type wayland.
en~waylandError~Program only works with X11!
en~waylandError~Possible solution:
en~waylandError~  Displaymanagers gdm3 and sddm
en~waylandError~  often have an option to switch
en~waylandError~  to X11 on the logon screen:
en~waylandError~  gdm3 e.g. shows a wheel icon
en~waylandError~  at the right bottom corner.

en~vlcInitError~Error opening VLC Media Player
en~vlcInitError~
en~vlcInitError~Possible reasons can be:
en~vlcInitError~  All Operating Systems:
en~vlcInitError~    VLC Player ist not installed
en~vlcInitError~  Linux:
en~vlcInitError~    VLC was installed from Snap Store or
en~vlcInitError~    Flathub and runs in a sandbox, where
en~vlcInitError~    the access to the VLC library is blocked.
en~vlcInitError~    Solution:
en~vlcInitError~      1. Uninstall VLC.
en~vlcInitError~      2. Open a terminal window.
en~vlcInitError~      3. Type: sudo apt install vlc
en~vlcInitError~         (Install VLC from the repository)
en~vlcInitError~      4. Restart CyberTelly
en~vlcInitError~      5. Give it another try ...

en~vlcProcessError~Error: VLC Player Process  does not respond
en~vlcProcessError~
en~vlcProcessError~The VLC library  libvlc  runs in a separate
en~vlcProcessError~process.  Actually  it cannot  be accessed:
en~vlcProcessError~Possibly VLC has crashed due to a malformed
en~vlcProcessError~video stream.
en~vlcProcessError~
en~vlcProcessError~  >    Please quit program and    <
en~vlcProcessError~  >    restart it again!          <
