de~Tastenkürzel: 
de~  Strg-Taste entspricht Cmd-Taste auf MacOS
de~  Strg-P ......... = Programmliste öffnen
de~                     Programm streamen mit
de~                     Doppelkick oder Return
de~  P .............. = Streaming starten
de~  S .............. = Streaming stoppen
de~  Strg-V ......... = Vollbild ein
de~  ESC ............ = Vollbild aus
de~  Strg-T ......... = Toolbar anzeigen/verstecken
de~  Strg-9 ......... = Bildformat 16:9 einstellen
de~  Strg-L ......... = Lautstärke einstellen
de~  Leertaste ...... = Audio aus/ein
de~  Pfeil nach oben  = Audio lauter
de~  Pfeil nach unten = Audio leiser
de~  Strg-E ......... = Einstellungen-Dialog öffnen
de~  F1 ............. = Programmhilfe öffnen
de~  Strg-I ......... = Programminfo anzeigen
de~
de~Mausbedienung des Programms:
de~  Kontextmenü öffnen:
de~    Cursorposition im Programmfenster
de~    Rechte Maustaste
de~  Fenster verschieben:
de~    Cursorposition im Programmfenster
de~    Linke Maustaste + Drag & Drop
de~  Fenstergröße ändern:
de~    Cursorposition Fensterecke rechts unten
de~    Linke Maustaste + Drag & Drop
de~  Vollbild ein-/ausschalten
de~    Cursorposition im Programmfenster
de~    Doppelklick
de~  Lautstärke regeln (lauter/leiser)
de~    Cursorposition im Programmfenster
de~    Mausrad drehen
de~  EPG anzeigen (nur TVHeadend):
de~    Programmliste öffnen
de~    Cursor zu Sender bewegen
de~    Kurze Pause: Tooltip wird angezeigt
de~    Tooltip enthält 4 Einträge

de~Tipps zur Einrichtung der TV-Umgebung:
de~  Download fertiger IPTV-M3u-Playlists siehe:
de~    https://github.com/jnk22/kodinerds-iptv
de~    https://github.com/iptv-org/iptv (PLAYLISTS.md)
de~    Speicherort: 
de~      Ordner CyberTelly/m3u im Userverzeichnis
de~    Downloads aus den o.g. Websites können nicht
de~    lizensierte Streaming-Quellen enthalten. Die 
de~    Nutzung erfolgt auf eigene Verantwortung!
de~  Zusammenstellen einer eigenen IPTV-Playlist:
de~    Voraussetzung: Internet-Download Playlists
de~    Vorgehensweise:
de~      M3u-Playlist im Dateimanager / Finder suchen 
de~      VLC-Player öffnen
de~      Windows/Linux: Menü Ansicht-Wiedergabeliste
de~      Playlist in die Medienbibliothek ziehen
de~      M3u-Dateien in die Wiedergabeliste ziehen
de~      Wiedergabeliste öffnen
de~      Nicht benötigte Sender löschen (Entf)
de~      Gewünschte Sender umsortieren (Drag&Drop)
de~      Windows/Linux:
de~        Rechtsklick Playlist abspeichern
de~      MacOS: 
de~        Menü Ablage - Wiedergabeliste speichern
de~    Speicherort: 
de~      Ordner CyberTelly/m3u im Userverzeichnis
de~  Fritzbox-Cable - M3u-Liste erzeugen:
de~    Vorgehensweise:
de~      VLC-Player öffnen
de~      Windows/Linux: Ansicht-Wiedergabeliste
de~      Klick auf Universal Plug'n'Play
de~        Medienserver werden angezeigt
de~        Doppelklick öffnet Programmliste
de~      Drag & Drop: Sender > Wiedergabeliste
de~      Wiedergabeliste öffnen
de~      Drag & Drop: Sender umsortieren
de~      Windows/Linux:
de~        Rechtsklick Playlist abspeichern
de~      MacOS: 
de~        Menü Ablage - Wiedergabeliste speichern
de~    Speicherort: 
de~      Ordner CyberTelly/m3u im Userverzeichnis
de~  Sat>IP-Server - M3u-Liste erzeugen:
de~    Bei Sat>IP-Servern können Sender - ähnlich
de~    wie bei einer FritzBox-Cable - per M3u-Liste
de~    abgespielt werden. Fertige M3u-Listen siehe:
de~    https://github.com/dersnyke/satipplaylists
de~    Nach dem Download muss in der Playlist mit 
de~    einem Texteditor 'sat.ip' noch ersetzt werden
de~    durch die IP-Adresse des Sat>IP-Servers.
de~    Beispiel: sat.ip > 192.168.178.230
de~    Anschließend kann die Liste angepasst und
de~    gespeichert werden wie oben beschieben.
de~    Das Verfahren wurde mit einem Sat>IP-Server
de~    der Firma Megasat erfolgreich getestet.
de~  Infos zum TVHeadend-Server:
de~    Hardware: z.B. Raspi 4 mit Raspi OS lite
de~    Zugriff auf TVHeadend-Server:
de~      Url: <Protokoll><Server-IP>:<Port>
de~      Protokoll: http://
de~      Server-IP: IPv4-Adresse des Servers
de~      Port.....: 9981 (Voreinstellung TVHeadend)
de~      Beispiel.: http://192.168.178.235:9981
de~      Das Beispiel kann als Vorlage zur Eingabe
de~      in den CyberTelly-Einstellungen dienen.
de~    User-Einstellungen TVHeadend:
de~      Configuration - Users - Access Entries
de~      Configuration - Users - Passwords
de~    Authentifizierungs-Einstellungen TVHeadend:
de~      Configuration - General - Base
de~        Http Server Settings
de~          Authentication type: Plain (insecure)
de~          Digest hash type:    MD5
de~    Hinweis:
de~      CyberTelly funktioniert nur mit korrekten
de~      User- und Authentifizierungs-Einstellungen!
de~  Info zum EPG:
de~    EPG funktioniert nicht mit M3u-Playlists.

en~Keyboard Shortcuts:
en~  Ctrl Key is Cmd Key on MacOS
en~  Ctrl-P ... = Open channel list
en~               Double Click or Enter starts
en~               streaming selected channel
en~  P ........ = Start streaming
en~  S ........ = Stop streaming
en~  Ctrl-V ... = Switch to Fullscreen
en~  ESC ...... = Switch back from Fullscreen
en~  Ctrl-T ... = Show / hide Toolbar
en~  Ctrl-9 ... = Set Aspect Ratio to 16:9
en~  Ctrl-L ... = Set Volume
en~  Space Bar  = Audio muted / unmuted
en~  Arrow up . = Volume up
en~  Arrow down = Volume down
en~  Ctrl-E ... = Open Settings Dialog
en~  F1 ....... = Show Help Dialog
en~  Ctrl-I ... = Show About Dialog
en~
en~Mouse Control:
en~  How to open the context menu:
en~    Position cursor inside the window
en~    Click right mouse button
en~  How to move the window:
en~    Position cursor inside the window
en~    Left mouse button + Drag & Drop
en~  How to resize the window:
en~    Move cursor to the right bottom corner
en~    Left mouse button + Drag & Drop
en~  How to turn fullscreen on or off:
en~    Position cursor inside the window
en~    Double click left mouse button
en~  How to adjust volume quickly:
en~    Position cursor inside the window
en~    Turn mouse wheel
en~  How to display EPG (TVHeadend only):
en~    Open channel list
en~    Move cursor to the desired channel
en~    After a short delay tooltip pops up
en~    Tooltip has four entries

en~How to set up your TV environment:
en~  Download urls for IPTV m3u playlists:
en~    https://github.com/jnk22/kodinerds-iptv
en~    https://github.com/iptv-org/iptv (PLAYLISTS.md)
en~    Where to save the playlists: 
en~      Folder CyberTelly/m3u in user directory
en~    Downloads from the above websites can contain
en~    unlicensed streaming sources. Installing it is 
en~    at the users sole responsibility!
en~  How to create your own IPTV playlist:
en~    Make sure you have downloaded m3u lists
en~    How to build your own list:
en~      Open downloaded list in file manager / finder
en~      Open VLC Player
en~      Windows/Linux: View Playlist
en~      Drag playlist into media library
en~      Drag m3u files into playlist window
en~      Open Playlist window
en~      Delete channels you don't like (Del)
en~      Rearrange channel order (Drag & Drop)
en~      Windows/Linux:
en~        Right click: Save playlist
en~      MacOS:
en~        File - Save Playlist...
en~    Where to save your playlist: 
en~      Folder CyberTelly/m3u in user directory
en~  Fritzbox-Cable - How to create m3u list:
en~    Open VLC media player
en~    Windows/Linux: View-playlist
en~    Click on Universal Plug'n'Play
en~      Media servers show up
en~      Double click opens channel list
en~    Drag & Drop: Channels > Playlist
en~    Open Playlist window
en~    Drag & Drop: Rearrange channels
en~    Windows/Linux:
en~      Right click: Save playlist
en~    MacOS:
en~      File - Save Playlist...
en~    Where to save the playlist: 
en~      Folder CyberTelly/m3u in user directory
en~  Sat>IP Server - How to create m3u list:
en~    M3u playlists also work with Sat>IP servers,
en~    similar to a FritzBox-Cable. Almost ready
en~    to use m3u playlists can be downloaded from:
en~    https://github.com/dersnyke/satipplaylists
en~    After the download open the list with a 
en~    text editor and replace 'sat.ip' with the
en~    ip address of the server.
en~    Example: sat.ip > 192.168.178.230
en~    If you like you can customize and store 
en~    your playlist as described above.
en~    This procedure was successfully tested
en~    with a Megasat Sat>IP server.
en~  TVHeadend Server HowTo:
en~    Hardware: e.g. Raspi 4 with Raspi OS lite
en~    How to access the THV-Server:
en~      Url: <Protocol><Server IP>:<Port>
en~      Protocol.: http://
en~      Server IP: IPv4-Address of TVH server
en~      Port.....: 9981 (TVHeadend default)
en~      Example..: http://192.168.178.201:9981
en~      The example can be used as a blue print
en~      for the input in the settings dialog.
en~    User Settings TVHeadend:
en~      Configuration - Users - Access Entries
en~      Configuration - Users - Passwords
en~    Authentication Settings TVHeadend:
en~      Configuration - General - Base
en~        Http Server Settings
en~          Authentication type: Plain (insecure)
en~          Digest hash type:    MD5
en~    Please note:
en~      CyberTelly only works with correct user
en~      and authentication settings!
en~  EPG:
en~    EPG doesn't work with m3u playlists.

en~  Known issues:
en~    * Firewall blocks vdeo stream

en~    * Windows 11: VLC crashes because of a

en~    * Fedora 43: VLC crashes because of a
en~      conflict with the audio driver
